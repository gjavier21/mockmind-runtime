# MockMind Runtime

A modular Node.js cognitive engine with Markdown-native artifacts, threaded memory, semantic embeddings, auto-tagging, mirror-agent reflection, compression, and cross-thread synthesis.

This runtime implements the operational core of the MockMind Method.

---

## ✨ Features

- 🧵 Threaded memory
- 📝 Markdown-native artifacts
- 🧠 Semantic embeddings (OpenAI)
- 🏷️ Auto-tagging
- 🔍 Smart retrieval (semantic + recency + tags)
- 🪞 Mirror-agent reflection (tone + drift)
- 🧬 Cross-thread synthesis
- 🧠 Cognitive REPL (`mockmind-loop`)
